# สถานะเบอร์โทรศัพท์ (Phone Number Status)

ไฟล์นี้อธิบายสถานะต่างๆ ของเบอร์โทรศัพท์ในระบบ Supernumber เพื่อให้แอดมินและนักพัฒนาเข้าใจตรงกัน

## รายการสถานะ

| สถานะ (Technical) | ชื่อภาษาไทย | คำอธิบาย | การแสดงผลหน้าเว็บ |
| :--- | :--- | :--- | :--- |
| **`active`** | **ว่าง / พร้อมขาย** | เบอร์ปกติที่ยังไม่มีใครจองหรือซื้อ | แสดงในหน้า Catalog ให้ลูกค้าเลือกซื้อได้ |
| **`hold`** | **จอง / รอตรวจสอบ** | เบอร์ที่ติดจองจากการสั่งซื้อ แต่อยู่ระหว่างรอตรวจสลิปหรือดำเนินการ | ไม่แสดงใน Catalog (ถูกซ่อนไว้ชั่วคราว) |
| **`sold`** | **ขายแล้ว** | เบอร์ที่ขายออกไปเรียบร้อยแล้ว | ไม่แสดงใน Catalog และไม่สามารถนำกลับมาขายใหม่ได้ |

## การเปลี่ยนสถานะอัตโนมัติ
ระบบจะเปลี่ยนสถานะเบอร์อัตโนมัติตามสถานะของ **ใบสั่งซื้อ (Order)** ดังนี้:
- เมื่อมีการสั่งซื้อเข้ามาใหม่ -> เปลี่ยนเบอร์เป็น **`hold`**
- เมื่อแอดมินยืนยันว่าขายสำเร็จ (`completed`) -> เปลี่ยนเบอร์เป็น **`sold`**
- เมื่อรายการสั่งซื้อถูกยกเลิกหรือปฏิเสธ (`cancelled`) -> เปลี่ยนเบอร์กลับเป็น **`active`**
