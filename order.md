# สถานะคำสั่งซื้อ (Customer Order Status)

ไฟล์นี้อธิบายสถานะของใบสั่งซื้อในระบบ Supernumber ซึ่งได้รับการปรับปรุงให้เรียบง่ายขึ้น (Simplify) เพื่อความสะดวกในการจัดการ

## รายการสถานะที่ปรับปรุงใหม่

เพื่อให้แอดมินจัดการงานได้รวดเร็ว เราได้จัดกลุ่มสถานะเดิม 8 อย่าง ให้เหลือเพียง **3 กลุ่มหลัก** ดังนี้:

### 1. กำลังดำเนินการ (Processing)
- **Technical Status:** `processing` (ครอบคลุม `submitted`, `pending_review`, `reviewing`, `paid`)
- **ความหมาย:** รายการใหม่ที่เพิ่งส่งเข้ามา หรือแอดมินกำลังตรวจสอบเอกสาร/สลิปโอนเงิน
- **ผลต่อเบอร์:** เบอร์จะถูกล็อคสถานะเป็น **`hold`** (จอง)

### 2. สำเร็จ (Completed)
- **Technical Status:** `completed` (ครอบคลุม `completed`, `sold`)
- **ความหมาย:** แอดมินตรวจสอบสลิปเรียบร้อย และจัดส่งซิมหรือดำเนินการให้ลูกค้าเสร็จสิ้น
- **ผลต่อเบอร์:** เบอร์จะถูกเปลี่ยนสถานะเป็น **`sold`** (ขายแล้ว)

### 3. ยกเลิก (Cancelled)
- **Technical Status:** `cancelled` (ครอบคลุม `rejected`, `cancelled`)
- **ความหมาย:** รายการที่ถูกยกเลิกโดยลูกค้า หรือแอดมินปฏิเสธเนื่องจากข้อมูลไม่ถูกต้อง
- **ผลต่อเบอร์:** เบอร์จะถูกปลดล็อคกลับไปเป็น **`active`** (พร้อมขาย) เพื่อให้คนอื่นซื้อได้

---

## ตารางเปรียบเทียบ (Mapping)

| สถานะใหม่ | สถานะเดิมที่ครอบคลุม | ผลต่อสถานะเบอร์ |
| :--- | :--- | :--- |
| **กำลังดำเนินการ** | `submitted`, `pending_review`, `reviewing`, `paid` | **`hold`** |
| **สำเร็จ** | `completed`, `sold` | **`sold`** |
| **ยกเลิก** | `rejected`, `cancelled` | **`active`** |
